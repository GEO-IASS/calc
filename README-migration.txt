Before installing Calc 2:

	1. Delete System Environment variable CALC_HOME
	2. Clear your default Browser cache
	3. With your current user, open R/RStudio and install the packages RPosrgreSQL and sqldf:
		install.packages('RPostgreSQL')
		install.packages('sqldf')
	
During the installation:
	Make sure that the current user has read/write permissions of the folder where you are going to install Calc.
	(It is suggested to install Calc2 in a different folder than Calc1)
